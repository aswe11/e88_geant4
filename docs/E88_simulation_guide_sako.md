# E88 Simulation Guide

This document summarizes the environment, parameters, and procedures for the E88 simulation.

## 1. Remote Server Access
The code compiles and runs only on specific RIKEN CCJ servers due to dependencies on older compilers and libraries.

1.  SSH to the gateway: `ssh ccjgw.riken.go.jp`
2.  SSH to the compute node: `ssh ccjphi0306`

## 2. Directory Path
-   **Main Directory:** `/ccj/u/sako/E16G4E88`

## 3. Environment Setup
To set up necessary environment variables, run:
```csh
source E16setup.csh
```

## 4. Parameter and Geometry Files
-   **Config File:** `E16CalibFiles-local.cfg` (Defines geometry and parameter files used in the code).
-   **Geometry Files:**
    -   `geometry_202304_run0e.dat`: Defines the correct RPC geometry (rough approximation).
    -   `geometry_202505_run1.dat`: RPC at far position ($R \approx 1700\,\text{mm}$); includes Module 206 as configured in April 2024.

## 5. Executables
-   **`E16G4-yok.stat`**: GEANT4 executable used for physics analysis.
-   **`E16G4.vis-stat`**: GEANT4 executable for visualization.
    -   Use with macro: `./macro/vis-OGL.mac`

## 6. Build Instructions
To build the project from source:
```bash
make clean
make
```

## 7. Source Code Structure
-   **Source Files:** `./src/`
-   **Header Files:** `./include/`
-   **Analysis Code:** `./root/`

## 8. Execution and Analysis Procedure
Follow these steps to run the simulation and process the output:

1.  **Execute GEANT4:**
    ```bash
    ./E16G4-yok.stat ./macro/pion-simple.mac <geant4-input-file> <geant4-out-file>
    ```
2.  **Convert Binary to ROOT Tree:**
    ```bash
    ./g4out2tree <geant-out-file> <raw-root-file>
    ```
3.  **Convert Raw Tree to DST:**
    ```bash
    ./root/e16anae88 <raw-root-file> <ana-out-file1>
    ```
4.  **Final Analysis:**
    ```bash
    ./root/ttreeanae88 ./ana-out-file1 ./ana-out-file2
    ```
